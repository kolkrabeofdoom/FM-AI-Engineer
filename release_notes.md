# Release v3.1.0: The Retro FX & QoL Update! 🎛️📺

Following the massive v3.0.0 WASM engine integration, this update focuses entirely on "Quality of Life" improvements and cranking the 80s aesthetic up to eleven!

### 🚀 New Features & Enhancements
* 📺 **Retro CRT / Scanline Mode**: A brand new toggle in the header that adds an authentic, flickering CSS-based CRT overlay with scanlines and vignettes to the entire application.
* ⌨️ **QWERTY Keyboard Support**: You can now play the virtual keyboard directly using your computer's QWERTY/QWERTZ keyboard (lower row = C3 octave, upper row = C4 octave) for lightning-fast melody testing.
* 🎲 **"Surprise Me" Prompt Generator**: Stuck in a creative rut? Click the new dice icon to instantly insert highly creative, poetic sound descriptions into the AI prompt field.
* 📟 **Authentic 16x2 LCD Display**: The patch name is no longer just text; it's now displayed inside a beautiful, glowing, neo-green 16x2 dot-matrix style LCD box, alongside the active Algorithm and Feedback values.
* 🎛️ **80s Stereo FX Chain (Chorus & Reverb)**: The DX7 WASM engine is mono, but now you can toggle the "80s FX" button to route the signal through a custom WebAudio Stereo-Chorus (modulated delay) and a lush algorithmic Reverb. Pads have never sounded wider!

### 🛠️ Technical Changes
* App version bumped to `3.1.0`
* `Documentation.tsx` (In-App Handbuch) fully updated to reflect v3.0 and v3.1 features.
* `README.md` updated with the new feature lists.
* Re-calibrated the `HexterNode` audio routing to support a parallel Dry/Wet processing chain.

Enjoy the ultimate retro FM synthesis experience right in your browser! 🎹✨
